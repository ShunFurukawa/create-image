# Create Sample Images


## Overview

### Description

bmp gif jpeg jpg png tiff tif Create a sample image with each file extension.

### Usage

```
$ sh create-image.sh width-size height-size
```

#### Example

```
$ sh create-image.sh 1500 1500

$ tree
.
├── create-image.sh
└── images
    ├── 1500x1500.bmp
    ├── 1500x1500.gif
    ├── 1500x1500.jpeg
    ├── 1500x1500.jpg
    ├── 1500x1500.png
    ├── 1500x1500.tif
    └── 1500x1500.tiff
```
